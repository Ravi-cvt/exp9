# Todo App with Docker CI/CD

Repo: https://github.com/Ravi-cvt/exp9

## Local Run
```bash
docker compose up
```
Frontend: http://localhost:3000  
Backend: http://localhost:3001/todos

## CI/CD
- **CI**: Tests, lint, Docker build on push/PR.
- **CD**: Docker build/push to Docker Hub (ravi-cvt/frontend-app:backend-app:latest) on main push.
